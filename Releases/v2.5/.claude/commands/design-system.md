# Design System Command

## Purpose
Create comprehensive design systems with components, tokens, and documentation for consistent user experiences.

## Agent Assignment
**Primary Agent:** `~/.claude/agents/designer.md`

## Workflow
1. Design audit and component inventory
2. Design token definition and standardization
3. Component library creation
4. Documentation and usage guidelines

## Output
- Complete design system specification
- Component library with variants
- Design token definitions
- Implementation guidelines
