# Newest Fabric Patterns Command

## Purpose
Discover and analyze the latest Fabric patterns using web search to find recent releases and updates, explaining the features and benefits of each new pattern.

## Agent Assignment
**Primary Agent:** `~/.claude/agents/developer.md`
**Supporting Agent:** `~/.claude/agents/writing.md` (for explanations)

## Workflow
1. Web Search for Latest Patterns
2. Pattern Analysis
3. Report Generation

## Output
- Summary report with the three newest patterns
- Individual pattern analysis with features/benefits
- Usage examples for each pattern
- Release information and dates
- Integration recommendations
