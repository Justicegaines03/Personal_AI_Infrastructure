# Security Audit Command

## Purpose
Comprehensive security assessment of applications, infrastructure, and processes with actionable remediation guidance.

## Agent Assignment
**Primary Agent:** `~/.claude/agents/pentester.md`

## Workflow
1. Vulnerability scanning and assessment
2. Security control evaluation  
3. Risk analysis and prioritization
4. Remediation planning and implementation guidance

## Output
- Detailed security assessment report
- Prioritized vulnerability list
- Remediation roadmap
- Compliance checklist
