# User Story Mapping Command

## Purpose
Create comprehensive user story maps with personas, journeys, and feature prioritization for product development.

## Agent Assignment
**Primary Agent:** `~/.claude/agents/designer.md`
**Supporting Agent:** `~/.claude/agents/marketer.md`

## Workflow
1. User persona development and validation
2. User journey mapping and analysis
3. Story creation and prioritization
4. Feature roadmap development

## Output
- User story map with prioritization
- Persona profiles and journey maps
- Feature backlog and roadmap
- Acceptance criteria definitions
