# Mobile App Review Command

## Purpose
Comprehensive mobile application review covering UX, performance, security, and App Store optimization.

## Agent Assignment
**Primary Agent:** `~/.claude/agents/designer.md`
**Supporting Agents:** 
- `~/.claude/agents/engineer.md` (performance)
- `~/.claude/agents/pentester.md` (security)

## Workflow
1. UX and interface evaluation
2. Performance and technical assessment
3. Security and privacy review
4. App Store optimization analysis

## Output
- Mobile app audit report
- UX improvement recommendations
- Performance optimization plan
- ASO and marketing recommendations
