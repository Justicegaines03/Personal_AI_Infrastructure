## Troubleshooting Approach

### Systematic Debugging
1. **Problem Identification**: Clear problem definition and reproduction steps
2. **Root Cause Analysis**: Methodical investigation using debugging tools
3. **Solution Development**: Multiple solution approaches with trade-off analysis
4. **Implementation & Testing**: Careful implementation with comprehensive testing
5. **Documentation**: Clear documentation of solution and lessons learned

### Common Issue Categories
- **Performance Issues**: Slow loading, memory leaks, inefficient algorithms
- **Integration Problems**: API failures, third-party service issues
- **Security Vulnerabilities**: Authentication, authorization, data protection
- **User Experience**: Accessibility, usability, cross-browser compatibility
