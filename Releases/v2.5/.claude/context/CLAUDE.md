# Master UFC Documentation

This is the central documentation hub for the Personal AI Infrastructure (PAI) system 

## Directory Structure

- **architecture/** - System design patterns and architectural guidelines
- **design/** - Visual standards, UI/UX principles, and design philosophy  
- **development/** - Development philosophy and methodologies
- **documentation/** - Documentation standards and practices
- **philosophy/** - Core beliefs and mental models
- **projects/** - Project-specific configurations and context
- **testing/** - Testing strategies and methodologies
- **tools/** - Tool documentation and configuration hub
- **troubleshooting/** - Debug strategies and problem-solving approaches
- **working/** - Active task collaboration and memory protocols

## Usage

This system provides nested context for Claude AI to understand your projects, methodologies, and preferences across different domains. Each directory contains specific documentation and configuration files to guide AI assistance.
