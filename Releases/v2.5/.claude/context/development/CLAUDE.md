## Tools and Frameworks

Your preferred tools for developing are Node.js for websites and React Native for Mobile Apps.

###Node.js Overview

Node.js is used for server-side development, such as building APIs, handling database operations, and backend services.​

It runs JavaScript outside the browser, using the powerful V8 engine, and is popular for high-performance, scalable network applications.​

Suitable for projects that demand real-time operations, large-scale networking, or custom server logic.​

Documentation can be found here: https://nodejs.org/docs/latest/api/

###React Native Overview

React Native is a framework to develop cross-platform mobile apps on iOS and Android from a single codebase using React and JavaScript.​

It is strictly used for front-end mobile UI development, providing native-like experience and performance for apps.​

Ideal for teams needing fast mobile development while reusing code across platforms.

Documentation can be found here: https://reactnative.dev/docs/getting-started

###N8N Overview
n8n is a workflow automation platform that allows users to connect different apps, services, and data sources through visual, node-based workflows. 

This makes it possible to automate both simple and complex business processes without deep coding knowledge. 

It stands out for its flexibility, extensibility, and data privacy focus, offering both cloud and robust self-hosted deployment options as well as seamless code integration for advanced customizations.

Documentation can be found here: https://docs.n8n.io/
