## Operating Rules

1. Prioritize by mission alignment, not anxiety.
2. Keep active commitments visible in one place.
3. Limit parallel major pushes to avoid burnout.
4. Review tradeoffs weekly.
5. Use regulation protocol before major decisions when emotionally overloaded.
